Excelator

Generate an Excel readable XML file

Download this project as a .zip file Download this project as a tar.gz file

This gem

Generates an XML file, that can be interpreted as XLS when you use it with MS Excel.

Install

  $ gem install excelator

Usage

Copie lo que sigue a su programa: En la primera instrucci√≥n del programa que va a utilizar esta yema colocar 

  require 'excelator'

Crea el libro excel de salida "libro.xml"

  f=Excelator.new("libro")

Inicia el libro con los encabezados de excel y los estilos a utilizar

  f.iniciar

Luego se definen las planillas del libro:

Inicia una planilla en el libro (hay que hacerlo para cada planilla

  f.iniciar_planilla("planilla")

Hay que hacer uno para cada columna con el ancho 9999 medido en caracteres, todas se definene una debajo de la otra 

  f.columna(ancho)

Luego se definen las lineas

  f.iniciar_linea

Hay que enviar las celdas en el orden que aparecen en la linea. Tipos: Encabezado, GranTitulo, Titulo, Alfa, Num, Fecha, Total (Si en el contenido aparece un signo igual lo considera una formula)

  f.celda("tipo","contenido")           

  f.terminar_linea

Para terminar f.terminar_planilla #Termina la planilla en curso del Libro f.terminar
